# Post Management Backend API

A simple RESTful backend API built using **Node.js** and **Express.js** that allows users to manage posts.  
The project includes CRUD operations for posts and a mock login API for authentication simulation.

---

## 🚀 Features

- Get all posts
- Get post by ID
- Add new post
- Update post by ID
- Delete post by ID
- Mock user login API
- RESTful API structure
- JSON request & response

---

## 🛠️ Tech Stack

- Node.js
- Express.js
- JavaScript
- REST API
